# Quant Research Task

This skill is deprecated.

Its useful responsibilities have been absorbed by:

- `../../fundamental/file-organization/SKILL.md` for project-level folder organization, research documents, project nodes, data, notebooks, outputs, and skills.
- `../../fundamental/exploration-task-workflow/SKILL.md` for question discovery, uncertain research direction, assumption discovery, option comparison, and evidence-backed next steps.

Do not use this skill as the primary workflow for new quantitative research projects. If an old project still references `quant-researcher` or `quant-research-task`, migrate the workflow to File Organization plus Exploration Task Workflow.

## Historical Scope

The deprecated workflow organized quantitative research around question-oriented folders such as `questions/`, `research/`, `reports/`, and concrete project/subtask folders.

Keep this file only as a compatibility marker so agents understand that this quantitative research task workflow was intentionally deprecated rather than accidentally omitted.
